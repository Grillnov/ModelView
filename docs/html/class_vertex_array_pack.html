Register a buffer object storing single precision floating points as an vertex attribute at layout index(layout location = index). Rejects the operation if the index is already occupied.

Parameters
bufferThe buffer storing attribute values.
indexThe layout index of the attribute in GLSL shader code.
componentsHow many components a vertex can have. A value between 1 to 4 is expected.
