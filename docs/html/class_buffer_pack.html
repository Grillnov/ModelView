Detailed Description

template<typename GLclientside>
class BufferPack< GLclientside >

Managing chunks of continuous memory on the server side as buffer objects. Generic class for all kinds of buffers. Note that this buffer class only serves as a shell injecting data into server side. There's not a copy at client side.

Parameters
GLclientsidethe client side types defined by OpenGL standard. Such as GLint, GLuint, GLfloat, GLdouble, etc.

When glBufferData is invoked, specified binding target and usage will affect the memory location allocated for this buffer on the server side. Use naiver Attach() is recommended when you're not so sure how to boost memory access performance by designating target and usage.

Parameters
hintBindedtargetThe GLenum target the buffer will be binded to when it's created. for example, GL_ARRAY_BUFFER, GL_ELEMENT_ARRAY_BUFFER, GL_COPY_WRITE_BUFFER, etc.
hintUsageThe GLenum usage the buffer will be used. for example, GL_STATIC_DRAW, GL_DYNAMIC_DRAW, etc.
